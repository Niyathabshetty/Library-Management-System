# Library Management System (Java)

## Description
This is a simple console-based Library Management System developed using Core Java. It allows users to manage books and users, issue and return books, and calculate fines for late returns.

## Features
- Add Books
- Add Users
- Issue Book
- Return Book with Fine Calculation
- View Books
- View Users

## Concepts Used
- OOP (Classes & Objects)
- ArrayList
- Loops & Conditions
- Methods
- User Input (Scanner)

## How to Run
1. Open project in Eclipse
2. Run `LibrarySystem.java`
3. Use menu options to interact
